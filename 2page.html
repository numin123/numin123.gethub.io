CSS Specificity

อะไรคือ Specificity?

หากมี CSS-rules ที่ขัดแย้งกันสองกฎขึ้นไปซึ่งชี้ไปที่ Element เดียวกัน เบราว์เซอร์จะทำตามข้อกำหนดบางอย่าง เพื่อพิจารณาว่ากฎข้อใดมีความ Specific มากก็จะชนะไป

ให้คิดว่า Specificity มีอันดับที่กำหนด style ว่าจะต้องใช้กับอะไร

The universal selector จะมี specificity ที่ต่ำกว่า ID selectors

Note: Specificity เป็นหนึ่งในเหตุผลที่ทำให้ CSS-rules ไม่สามารถใช้กับ Element ได้ในบางครั้ง อย่างที่ควรจะเป็น

Example

h1 {background-color: yellow;}

h1 {background-color: red;}

the latter rule is always applied.
