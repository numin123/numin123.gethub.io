Assignment 2 CSS Framework

CSS Specificity

อะไรคือ Specificity?

เป็นวิธีการที่เบราว์เซอร์ตัดสินใจว่าค่าคุณสมบัติ CSS ใดที่เกี่ยวข้องกับ Element มากที่สุด CSS ค่านั้นก็จะถูกนำไปใช้ หรือ ค่าบางอย่างที่จะจะบอก browser ว่าให้เลือกใช้ Style ตัวไหนมาแสดงผลหน้าเว็บนั่นเอง

ให้คิดว่า Specificity มีกฎที่กำหนด style ว่าจะต้องใช้กับอะไร

เช่น The universal selector จะมี specificity ที่น้อยกว่า ID selectors

Note: Specificity เป็นหนึ่งในเหตุผลที่ทำให้ CSS-rules ไม่สามารถใช้กับ Element ได้ในบางครั้ง อย่างที่ควรจะเป็น

Example

h1 {background-color: yellow;}

h1 {background-color: red;}

the latter rule is always applied.
