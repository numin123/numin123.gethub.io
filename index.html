Assignment 2 CSS Framework

1. ให้นิสิตสร้างหน้าเว็บอย่างน้อย 2 หน้า โดยมีลิงก์เชื่อมโยงถึงกัน (เมนู) หน้าแรก ให้ออกแบบหน้าเว็บที่บ่งบอกถึงนิสิตเอง (ระวังการนำข้อมูลส่วนตัวมาแสดงด้วย) หน้าที่สอง ให้ทำความเข้าใจและสรุปหลักการของ CSS Specificity พร้อมยกตัวอย่าง ในรูปแบบของหน้าเว็บอย่างสวยงาม หน้าที่สาม และอื่น ๆ เพิ่มเติมได้ ถ้าเนื้อหามีมากจนต้องแสดงหลายหน้า 2. ทุกหน้า ให้นำ CSS Framework มาใช้ โดยเลือก CSS Framework ที่ชอบ เพียง 1 Framework อาจเขียน external style sheet เพิ่มเติมเล็กน้อยได้ 3. นำ Font Icon มาใช้ด้วย อย่างเหมาะสม 4. push งานขึ้น github pages ของตนเอง 5. ส่ง URL ของ github pages ของตนเอง ที่สมบูรณ์ ทดสอบแล้วว่าเข้าได้จริง ใน Google Classroom และอย่าลืมกด Turn in ด้วย ไม่ตรวจงานที่ส่งเลยเวลา ทุกกรณี
